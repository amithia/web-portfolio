# Amithi Liyanagamage — Web Portfolio

Personal portfolio site showcasing work across UI/UX, Data & Analytics, AI Ethics, AI Education, Technology in Education, AI Policy, and Start-ups.

## Structure

```
/
├── index.html              # Main portfolio page
├── css/
│   ├── tokens.css          # Design tokens (colours, type, spacing)
│   ├── base.css            # Reset + global typography
│   ├── components.css      # Nav, hero, cards, buttons, footer
│   └── animations.css      # Scroll-in reveals, marquee, hero animations
├── js/
│   └── main.js             # Filter, scroll behaviour, intersection observer
├── pages/
│   └── case-study.html     # Case study template (to be built)
└── assets/
    ├── images/             # Portrait, project images
    └── icons/              # SVG icons
```

## Design System

**Theme:** Editorial Atlas — travel, cartography, art

**Colours:**
- Ink `#0D0D0D` · Parchment `#F4EFE6` · Saffron `#F5A623`
- Cobalt `#1B3FFF` · Terracotta `#C45C3A` · Forest `#2C5F2E`

**Fonts:** Clash Display · Space Grotesk · DM Serif Display · Inter
